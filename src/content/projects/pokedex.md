# Pokédex - Proyecto Personal

## Descripción del Proyecto

Este proyecto fue creado para aprender y dominar **Astro**, un framework moderno para construir sitios web rápidos con menos JavaScript del lado del cliente.

## Tecnologías Utilizadas

- **Astro 4.0**: Framework principal
- **PokeAPI**: API pública de Pokémon
- **Tailwind CSS**: Estilos
- **TypeScript**: Tipado fuerte

## Características

- Listado de todos los Pokémon
- Búsqueda y filtros
- Paginación
- Diseño responsive
